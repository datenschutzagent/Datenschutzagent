# Dokumentation

Die Dokumentation wurde in das **MkDocs**-Verzeichnis verschoben.

- **Quellen:** `mkdocs/docs/`
- **Konfiguration:** `mkdocs/mkdocs.yml`
- **Lokal anzeigen:** Aus dem Projektroot: `mkdocs serve -f mkdocs/mkdocs.yml`
- **Build:** `mkdocs build -f mkdocs/mkdocs.yml`

Siehe auch [README im Projektroot](../README.md#dokumentation).
